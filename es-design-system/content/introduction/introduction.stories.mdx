## Usage

To begin using the EnergySage design system packages in your own vue project:

```bash
$ npm install @energysage/es-bs-base @energysage/es-bs-extends @energysage/es-vue-base

# main.scss:
$primary: #000000; # Override default variables
@import '~@energysage/es-bs-extends/scss/variables.scss';
@import '~@energysage/es-bs-base/scss/bootstrap.scss';
# TODO: Should this be included by default?
@import '~@energysage/es-vue-base/scss/bootstrap-vue.scss';

# MyComponent.vue:
<template>
    <es-button>Submit</es-button>
</template>
<script>
import { EsButton } from '@energysage/es-vue-base'
export default {
    components: {
        EsButton,
    },
};
```
